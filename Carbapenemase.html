CIM grouping:

  • mCIM-eCIM- => no carbapenemase detected
  • mCIM+eCIM- => class A or D
  • mCIM-eCIM+ => lab error? should be rechecked
  • mCIM+eCIM+ => classB

MPCR & CARBA5 grouping:

  • 單一KPC或OXA：對應KPC/OXA
  • 單一NDM或VIM或IMP：對應IMP/NDM/VIM
  • 多樣偵測(e.g. KPC+OXA)：對應Others
  • 沒有偵測到：對應Negative
